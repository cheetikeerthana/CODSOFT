# CodSoft Java Programming Internship

## About the Internship
This repository contains the tasks completed as part of the **CodSoft Java Programming Internship**.  
The internship focuses on strengthening core Java concepts through hands-on, console-based projects.

Each task is designed to improve problem-solving skills and understanding of **Object-Oriented Programming (OOP)** using Java.

## Internship Tasks Overview

###  Task 1: Number Guessing Game
A console-based game where the system generates a random number and the user tries to guess it within a limited number of attempts.  
The game supports multiple rounds and displays the final score.

**Concepts Used:**
- Loops
- Conditional statements
- Random number generation
- Scanner class

Folder: 'Task-1'

###  Task 2: Student Grade Calculator
A Java program that calculates the total marks, average percentage, and grade of a student based on marks entered for multiple subjects.  
The program validates inputs and checks for subject-wise failure.

**Concepts Used:**
- Loops and conditions
- Input validation
- Basic arithmetic operations
- Scanner class

Folder: 'Task-2'
